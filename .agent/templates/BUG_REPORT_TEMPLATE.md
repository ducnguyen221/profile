## 2. PWA Cache Check (Bắt Buộc)
Hãy đảm bảo bạn đã thử điều này trước khi tạo bug:
- [ ] Bấm F12 -> Vô tab Application.
- [ ] Mục Storage -> Click **"Clear site data"**.
- [ ] Refresh lạị. Nếu lỗi hết, đây là lỗi do Dev quên update Version `CACHE_NAME`, VUI LÒNG update lại workflow SW chứ không phải sửa CSS.
